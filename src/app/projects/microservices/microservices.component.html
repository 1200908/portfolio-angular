Project Overview

A personal project built to demonstrate a robust, modular, and scalable microservices architecture using Java and Spring Boot. The system leverages centralized configuration management with Spring Cloud Config, dynamic service discovery via Eureka, and a unified API Gateway for streamlined request routing.

The project also integrates Spring Mail for automated notifications and employs architectural patterns such as CQRS and database-per-service to ensure modularity, autonomy, and clear data boundaries between services. Automated testing with JUnit reinforces reliability and maintainability.

Challenge: Coordinating multiple microservices with independent databases.

Configuration Management
