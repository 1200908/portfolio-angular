Project Overview

This project is a full-stack auto-generated blog built as a technical challenge. The frontend is built with React and communicates with a Node.js backend that stores articles in PostgreSQL. Both services are containerized using Docker and deployed on an AWS EC2 instance.

The backend generates articles automatically using a free AI API. The project demonstrates modern full-stack development practices, including containerization, CI/CD pipelines, and cloud deployment.

Challenges & Solutions

Automatic Article Generation

Challenge: Generating articles daily using an external AI API while maintaining data consistency.

Solution: Built a scheduled Node.js service that requests the AI API, saves articles in PostgreSQL, and ensures no duplicates.

Deployment & CI/CD

Challenge: Containerizing both frontend and backend and deploying with automated CI/CD on AWS.

Solution: Used Docker Compose for local setup, AWS ECR for container registry, and AWS CodeBuild for CI/CD pipeline to EC2.

Cross-Service Communication

Challenge: Frontend must fetch articles from backend efficiently and handle API errors gracefully.

Solution: Implemented Axios with proper error handling, loading states, and caching to improve UX.
