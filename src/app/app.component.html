Java Microservices Manager

Personal Project

Project designed to implement a robust and scalable Java-based microservices architecture. It leverages Spring Boot with Spring Cloud Config for centralized configuration management, Eureka Service Registry for dynamic service discovery, and an API Gateway to handle request routing and security. The system is containerized with Docker for isolated database instances and service deployment, ensuring reproducibility and ease of scaling. CQRS and database-per-service patterns are applied to maintain modularity and reliability across services, while automated testing with JUnit guarantees stability and maintainability.

  • Technologies: Java, Spring Boot, Spring Cloud Config, Eureka, Docker, JUnit
  • Highlights: Robust configuration management, scalable microservices, containerized deployment, modular design, automated testing
